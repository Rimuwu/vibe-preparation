# Essential English for IT Students

## Category 1. Computer Architecture & Hardware

### 1. How many generations of computers are there? What is the difference between them? Talk about each generation of computers in detail. #hardware #history
* **Краткий ответ**: The first generation used vacuum tubes; the second used transistors (10x faster); the third used integrated circuits (1M calculations/sec); the fourth used microprocessors (reduced size); and the fifth is based on parallel processing and AI.
* **Пояснение**: The first generation of computers used vacuum tubes. The second generation could work ten times faster because of the use of transistors instead of vacuum tubes. The third-generation computers could perform a million calculations per second and were controlled by tiny integrated circuits. In the fourth generation, integrated circuits were significantly reduced in size. The fifth generation, being developed nowadays, is based on parallel processing and Artificial Intelligence.

### 2. Talk about the contribution of Charles Babbage and John von Neumann to the development of computing technology. #history
* **Краткий ответ**: Charles Babbage designed the Analytical Engine (ALU, control unit, memory, I/O). John von Neumann proposed storing program instructions in the computer's own memory (stored-program concept).
* **Пояснение**: The analytical engine developed by Charles Babbage contained an ALU (arithmetic logic unit), a control unit, memory, and an input—output system - the main components of a modern computer. Another important achievement was that John von Neumann developed the idea of keeping computer instructions in the computer's own memory.

### 3. What do you know about Moore’s Law? #hardware #laws
* **Краткий ответ**: Moore’s Law states that computer processing power should double approximately every two years.
* **Пояснение**: Moore’s Law, which is based on various factors, states that computer processing power should double every two years.

### 4. What are the differences between the main types of computers (mainframes, supercomputers, desktop PCs, laptop PCs, tablet PCs, smartphones)? #hardware #types
* **Краткий ответ**: Mainframes process thousands of simultaneous queries; supercomputers run billions of operations per second on one complex problem; desktops are stationary; laptops are portable; tablets are handheld touchscreens; smartphones are smaller cellular-connected tablets.
* **Пояснение**: A mainframe can process and store large amounts of data. A supercomputer has a higher level of performance: it can perform billions of operations per second on one complex problem at a time, while a mainframe processes thousands of queries simultaneously. Desktops are stationary computers that stay in one place. Laptops are portable versions with a built-in keyboard and display. Tablets are highly portable devices with touchscreen, and smartphones are smaller tablets with cellular connections.

### 5. Talk about the CPU (the three main parts, the factors affecting the performance of the CPU). #cpu #hardware
* **Краткий ответ**: The CPU contains a control unit (directs operations), ALU (performs calculations), and registers (temporary storage). CPU performance is heavily affected by its clock speed (in GHz).
* **Пояснение**: The CPU consists of three main parts: the control unit that directs operations, the ALU that performs mathematical calculations and logical operations, and registers that provide high-speed temporary storage. CPU performance depends heavily on its clock speed, measured in gigahertz (GHz).

### 6. What is cache memory? What is ‘write-through’ cache and ‘write-back’ cache? What is cache hit? #cpu #memory
* **Краткий ответ**: Cache is an ultra-fast CPU buffer. Write-through updates cache and main memory together (safer but slower). Write-back updates cache first, writing to main memory later. A cache hit is finding data in the cache without reading main memory.
* **Пояснение**: Cache memory is a small, ultra-fast buffer inside the CPU that speeds up data access from the slower main memory. In write-through mode, data is updated in both the cache and main memory simultaneously, which is safer but slower. In write-back mode, data is written only to the cache first, updating main memory later when the cache block is replaced. A cache hit happens when the CPU finds the required data directly in the cache, avoiding a slow main memory read.

### 7. What is ROM? What is stored in ROM? Talk about different types of ROM and their distinctive features. #memory #hardware
* **Краткий ответ**: ROM is non-volatile memory storing basic CPU instructions. Types include MROM (factory programmed), PROM (programmable once), EPROM (UV-erasable), EEPROM (electrically erasable), and Flash ROM (fast EEPROM).
* **Пояснение**: ROM (read-only memory) is non-volatile memory that stores basic CPU instructions. It has several types: MROM is permanently coded during factory manufacturing; PROM is a clean chip that can be programmed only once; EPROM can be erased using ultraviolet light and rewritten; EEPROM can be erased and reprogrammed electrically; and Flash ROM is a modern, significantly faster version of EEPROM with very high endurance.

### 8. What is RAM? What data is stored in RAM? Talk about the two types of RAM and their distinctive features. #memory #hardware
* **Краткий ответ**: RAM is volatile memory storing active OS, app, and CPU data. The two main types are SRAM (static, fast, no refresh, used for cache) and DRAM (dynamic, needs refresh, cheaper, used for main memory).
* **Пояснение**: RAM (random access memory) is volatile memory that temporarily stores the operating system, running applications, and active data currently used by the CPU. The two main types are SRAM (Static RAM) and DRAM (Dynamic RAM). Fast, non-refreshing SRAM is used for CPU cache, while cheaper, refreshing DRAM (like DDR4/DDR5) serves as main memory.

### 9. What are the differences between primary memory and secondary storage devices? Give examples of primary memory and secondary storage media. #memory #storage
* **Краткий ответ**: Primary memory is fast, volatile/non-volatile, and directly accessed by the CPU (e.g., RAM, cache). Secondary storage is slower, non-volatile, and has larger capacity for long-term storage (e.g., HDD, SSD).
* **Пояснение**: Primary memory is fast, directly accessed by the CPU, and includes volatile or non-volatile options like RAM and cache. Secondary storage (such as HDDs and SSDs) is slower but provides much larger capacity for long-term storage.

### 10. How can secondary storage be classified? Talk about each category of secondary storage devices you have named. #storage
* **Краткий ответ**: Classified by technology: magnetic (HDDs using magnetic platters), optical (CDs/DVDs using lasers), and solid-state (SSDs using flash memory). Also classified by location: internal or external.
* **Пояснение**: Secondary storage is classified by technology into magnetic (HDDs using magnetic patterns for high capacity), optical (CDs/DVDs using lasers), and solid-state (SSDs using electronic flash memory for speed and durability). It can also be categorized by location as either internal or external.

### 11. Talk about the HDD. Describe the construction of a hard disk drive and the process of recording data on it. What is disk fragmentation/ defragmentation? #storage #hdd
* **Краткий ответ**: HDD records data magnetically on spinning platters using read/write heads. Fragmentation splits files into scattered sectors, slowing the system. Defragmentation reorganizes them back into contiguous blocks.
* **Пояснение**: An HDD uses read/write heads to record data magnetically on the tracks and sectors of spinning platters. Fragmentation happens when files are scattered across non-contiguous sectors, slowing performance. Defragmentation reorganizes these fragments into contiguous sectors to restore speed.

### 12. What types of optical storage devices do you know? Talk about each of them. #storage #optical
* **Краткий ответ**: CD (700 MB for audio/basic data), DVD (4.7 GB with a shorter laser for video), and Blu-ray (25+ GB with blue-violet laser for HD). All store data as microscopic pits on a reflective layer read by laser.
* **Пояснение**: Optical storage includes CDs (700 MB for audio or basic data), DVDs (4.7 GB using a shorter laser for video), and Blu-ray discs (from 25 GB using a blue-violet laser for HD video). All of them store data as microscopic pits on a reflective layer, which are read using a laser beam.

### 13. What are the two types of flash memory? How do they differ in functionality and use cases? #storage #flash
* **Краткий ответ**: NOR and NAND. NOR flash offers random byte access (ideal for direct firmware execution). NAND accesses data in blocks (higher density, lower cost, used in SSDs, memory cards, and USBs).
* **Пояснение**: The two main types of flash memory are NOR and NAND. NOR flash allows random byte access, making it ideal for directly executing firmware. NAND flash accesses data in blocks, providing much higher density and lower costs, making it perfect for mass storage devices like SSDs, memory cards, and USB drives.

## Category 2. Operating Systems & Software

### 14. What is software? What is the difference between system software and application software? What types of system software can you name? #software
* **Краткий ответ**: Software is the programs running a computer. System software controls hardware/background processes (OS, drivers, utilities). Application software runs tasks for users (browsers, games).
* **Пояснение**: Software is the non-physical programs that run a computer. System software rules the hardware and background operations (like operating systems, drivers, and utilities). Application software runs on top of it, letting users do specific tasks like browsing the web or gaming.

### 15. What is an OS? What are its main functions? Name the main types of operating systems and provide the examples. #os
* **Краткий ответ**: An OS manages hardware, memory, files, security, and UI. Types include single-task (MS-DOS), multitasking (Windows, macOS), multi-user (Linux servers), real-time (embedded systems), and mobile (Android, iOS).
* **Пояснение**: An OS is software that manages computer hardware, memory, files, security, and the user interface. Its main types include single-task systems (like MS-DOS), multitasking systems (Windows, macOS), multi-user systems (Linux servers), real-time systems (medical or automotive embedded systems), and mobile operating systems (Android, iOS).

### 16. What are the two types of multitasking mentioned in the text? (Essential English for IT students. Part 1. Unit 6, text B) What are the differences between the two? #os #multitasking
* **Краткий ответ**: Preemptive and cooperative multitasking. In preemptive, the OS forcibly interrupts programs to share CPU time. In cooperative, programs must voluntarily yield control (a frozen app blocks everything).
* **Пояснение**: In preemptive multitasking, the OS forcibly interrupts programs to share CPU time. In cooperative multitasking, programs must voluntarily yield control, meaning one frozen application can block the entire system.

## Category 3. Computer Networks & Internet

### 17. What can be the criteria for network classification? Provide several classifications of networks. #networks
* **Краткий ответ**: Network classifications are based on: geographic scope (PAN, LAN, MAN, WAN), architecture (client-server vs. P2P), transmission medium (wired vs. wireless), and topology (star, bus, ring, mesh).
* **Пояснение**: Networks are classified by four main criteria: geographic scope (PAN, LAN, MAN, WAN), architecture (client-server vs. peer-to-peer), transmission medium (wired vs. wireless), and topology (the physical layout, such as star, bus, ring, or mesh).

### 18. What is a network topology? Describe different network topologies mentioning their benefits and downsides. #networks #topology
* **Краткий ответ**: Network topology is the connection layout. Bus uses one cable (cheap, but single failure point). Star connects to a hub (safe for single devices, but hub is single failure point). Ring is a closed loop (orderly, but single break stops it). Mesh has redundant paths (highly reliable, but complex/costly).
* **Пояснение**: Network topology is the layout of a network's connections. Bus uses a single cable (cheap, but risky). Star connects everyone to a central hub (safe for individual devices, but the hub can fail). Ring forms a closed loop (orderly, but a single break stops everything). Mesh connects devices via multiple redundant paths (highly reliable, but expensive and complex).

### 19. Enumerate the main hardware components of a network and name their functions. #networks #hardware
* **Краткий ответ**: Modems (convert ISP signals), routers (link networks via IP), hubs (broadcast data), switches (target specific devices), bridges (divide networks for bandwidth), gateways (translate protocols), repeaters (extend signal range).
* **Пояснение**: Network hardware includes modems to convert ISP signals, routers to link networks via IP addresses, hubs to broadcast data to all devices, switches to target specific devices, bridges to divide networks for better bandwidth, gateways to translate protocols between different networks, and repeaters to extend signal range.

### 20. Talk about the OSI model. Name the layers, talk about their functions and the protocols used at different layers. #networks #osi
* **Краткий ответ**: 7-layer model: Application (HTTP, FTP), Presentation (formatting/encryption), Session (connections), Transport (TCP, UDP), Network (IP routing), Data Link (MAC framing), Physical (raw bit hardware transmission).
* **Пояснение**: The 7-layer OSI model manages network data: Application handles user apps (HTTP, FTP), Presentation formats and encrypts data, Session controls connections, Transport ensures reliable delivery (TCP, UDP), Network routes packets using IP addresses, Data Link frames data using MAC addresses, and Physical transmits raw bits over hardware.

### 21. What types of wired internet connection do you know? Talk about each of them in detail. #networks #internet
* **Краткий ответ**: DSL (high-speed digital over copper phone lines), Cable (broadband over TV networks/coaxial), and Fiber-optic (extremely fast light pulse transmission over long distances in binary).
* **Пояснение**: The three main types of wired connections are DSL, Cable, and Fiber-optic. DSL is high-speed internet transmitting digital signals over standard copper telephone lines. Cable is reliable broadband transmitting data via television networks and coaxial cables to a modem. Fiber-optic is incredibly fast internet transmitting data as light pulses over long distances using a binary system.

### 22. Talk about the three types of DSL connection. What is the difference between DSL and dial-up connection? #networks #internet
* **Краткий ответ**: ADSL (asymmetric, fast download), SDSL (symmetric upload/download), and VDSL (very fast fiber/copper hybrid). DSL allows simultaneous phone and internet use, whereas dial-up blocked the phone line.
* **Пояснение**: The three main types of DSL connections are ADSL, SDSL, and VDSL. ADSL is the most common home version; it is “asymmetric” because download speeds are much faster than upload speeds. SDSL is a business-focused version that provides equal (symmetrical) upload and download speeds. VDSL is an upgraded version using a fiber-and-copper mix to provide much faster speeds than ADSL. The main difference between DSL and dial-up is that DSL allows you to use the internet and the phone at the same time, while dial-up blocked your phone line because it used the same frequency.

### 23. Talk about the three wireless internet access technologies. What are their benefits and downsides? What equipment is utilized? #networks #wireless
* **Краткий ответ**: Wi-Fi (connects local devices to wireless router via radio waves), Satellite (connects via dish/modem to orbit, high latency), and Cellular (connects to grid towers using different frequencies).
* **Пояснение**: The three wireless technologies are Wi-Fi, satellite, and cellular networks. Wi-Fi connects local devices to a wireless router using radio waves. Satellite connects via orbiting satellites using a dish and modem, which leads to delays due to distance. Cellular uses a grid of land towers that send data on different frequencies to connect mobile devices.

### 24. What is the difference between the Internet and the WWW? #networks #internet #www
* **Краткий ответ**: The Internet is the physical network of computers and cables. The World Wide Web (WWW) is the system of websites and documents built on top of that network.
* **Пояснение**: The Internet is the physical network of connected computers and cables, while the World Wide Web (WWW) is the information system of websites and documents built on top of this network.

### 25. Talk about the packet switching and circuit switching technologies. #networks #protocols
* **Краткий ответ**: Circuit switching sets a dedicated path between points first. Packet switching breaks data into independent packets. Types: Datagram (packets take any route) and Virtual Circuit (packets take a pre-established route).
* **Пояснение**: Circuit switching sets a dedicated, blocked path between two points before sending the entire message, which can waste bandwidth. Packet switching breaks data into smaller, independent packets that can take different routes. There are two Packets switching types: Datagram (packets travel independently via different routes) and Virtual Circuit (all packets travel sequentially along a pre-established path).

### 26. Talk about the three types of search engines and their distinctive features. What three steps do crawler-based search engines take before displaying results to the user? #networks #search
* **Краткий ответ**: Crawler-based (automated bots), Human-powered (manual editors), and Meta-search (aggregates other engines). Crawler steps: Crawling (bots scan links), Indexing (storing keywords), Ranking (sorting relevance).
* **Пояснение**: There are three search engine types: Crawler-based (bots automatically find and rank pages), Human-powered (manually categorized by editors), and Meta-search (aggregates results from other engines). Three crawler steps: Crawling (bots scan links for content), Indexing (storing data by keywords), and Ranking (sorting results by relevance).

## Category 4. Programming & Web Development

### 27. How can programming languages be classified? Provide examples and talk about the distinctive features of each type you have named. What is the difference between a compiler, an interpreter, and an assembler? #programming #languages
* **Краткий ответ**: Low-level (machine, assembly close to hardware) and high-level (human-readable, portable). Assembler translates assembly to machine code. Compiler translates whole program to file, while interpreter runs code line-by-line.
* **Пояснение**: Programming languages are classified as low-level and high-level. Low-level languages work closely with hardware and include machine and assembly languages. High-level languages use human-readable syntax, making programs easier to write, debug, and transfer between systems. An assembler translates assembly language into machine code. A compiler converts an entire high-level program into an executable file, while an interpreter translates and runs code line by line.

### 28. What is OOP? What are the three key features of OOP? Talk about each of them in detail. #programming #oop
* **Краткий ответ**: Object-Oriented Programming centers on objects. Three features: encapsulation (hiding data), inheritance (reusing parent class code), and polymorphism (different reactions to the same method).
* **Пояснение**: Object-Oriented Programming (OOP) organizes software around objects. Its three key principles are encapsulation (hiding data), inheritance (reusing code from parent classes), and polymorphism (objects react differently to the same method).

### 29. What are the three main tools used in web design? Talk about each of them. What are the advantages and disadvantages of using website builders? #web #design
* **Краткий ответ**: HTML (structures content), CSS (controls styling), JavaScript (adds interactivity). Website builders offer fast setup using templates, but suffer from limited customization and vendor lock-in.
* **Пояснение**: HTML, CSS, and JavaScript are the main web design tools. HTML structures content, CSS controls appearance, and JavaScript adds interactivity and dynamic behavior. Website builders allow you to quickly create websites using templates and visual tools, but their disadvantages include limited customization and vendor lock-in.

### 30. What is the difference between frontend and backend web development? Talk about SEO. What is the difference between on-page SEO and off-page SEO? #web #seo
* **Краткий ответ**: Frontend handles user interface, backend manages server logic and databases. SEO boosts organic traffic. On-page optimizes internal items (content, keywords, speed), off-page builds external links.
* **Пояснение**: Frontend development focuses on the user interface, while backend development controls server logic, databases, security, and application functionality. SEO boosts visibility for organic traffic. On-page SEO optimizes internal elements (content, keywords, speed), while off-page SEO builds external authority (backlinks).

## Category 5. Cybersecurity & Cryptography

### 31. What is malware? Talk about the damaging effect of ransomware, spyware, adware, a Trojan, a miner. #security #malware
* **Краткий ответ**: Malware harms systems/steals data. Includes viruses (need user action), worms (auto-spread), Trojans (backdoors), spyware (steals data), ransomware (encrypts/locks files for ransom), and miners (hijacks resource).
* **Пояснение**: Malware harms systems or steals data. It includes viruses, which need user action to activate; worms, which spread automatically over networks; Trojans, which hide inside safe-looking programs and open a backdoor for hackers; spyware, which secretly steals data; and ransomware, which locks files until a payment is made.

### 32. What are the four routines of a virus? Talk about the function of each of them. What is the difference between a virus and a worm? #security #malware
* **Краткий ответ**: Replication (copies itself), protection (hides), trigger (activates), and payload (damages). A virus requires a host file and user action to spread, whereas a worm is standalone and spreads automatically.
* **Пояснение**: A virus relies on four routines: replication copies it, protection hides it, trigger activates it, and payload causes the actual damage. Unlike a virus, which requires a host file and user action to spread, a worm is standalone and propagates automatically across networks.

### 33. What is cybercrime? Talk about the damaging effect of phishing, website defacement, piggybacking and salami shaving. #security #crime
* **Краткий ответ**: Digital crime including identity theft, phishing (deceptive stealing of passwords), DDoS (crashing servers), defacement, piggybacking (unauthorized network use), and salami shaving (stealing tiny sums repeatedly).
* **Пояснение**: Cybercrime includes identity theft (stealing accounts data for fraud), phishing (deceptive messages used to steal passwords), DDoS attacks (flooding servers to crash websites), cyberstalking (online harassment), and digital piracy (unauthorized copying of copyrighted material).

### 34. What is the difference between DoS and DDoS attack? How is DDoS attack performed? #security #attacks
* **Краткий ответ**: DoS comes from a single source; DDoS comes from multiple sources simultaneously, using a botnet of infected devices to flood the target server and crash it.
* **Пояснение**: The main difference is that a DoS attack comes from a single source, while a DDoS attack uses multiple sources simultaneously. A DDoS attack uses a botnet of infected devices to flood the target server with traffic and crash it.

### 35. What is a firewall? What can firewall filters be based on? #security #firewall
* **Краткий ответ**: A firewall blocks unauthorized access by filtering traffic based on IPs, domain names, protocols, programs, ports, and keywords.
* **Пояснение**: A firewall blocks unauthorized network access by filtering traffic based on IPs, domains, protocols, programs, ports, and keywords.

### 36. How does anti-virus software work? What is the difference between signature-based scanning and heuristic scanning? What is a signature? #security #antivirus
* **Краткий ответ**: Scans files. Signature-based checks known malware signatures (unique code patterns). Heuristic scanning analyzes behavior to detect new, unknown threats.
* **Пояснение**: Signature-based scanning checks files against known malware, while heuristic scanning analyzes behavior to detect new, unknown threats. A signature is a unique sequence of bytes that identifies a specific malware.

### 37. What is encryption? Talk about the difference between its two main types. What two methods of breaking a cipher are there? #security #cryptography
* **Краткий ответ**: Secures data. Symmetric uses one key for encryption/decryption; asymmetric uses a public/private key pair. Ciphers are broken by brute force (trying keys) or cryptanalysis (finding structural flaws).
* **Пояснение**: Symmetric encryption uses one key for both tasks, while asymmetric uses public and private keys. Ciphers are broken via brute force (testing keys) or cryptanalysis (exploiting structural weaknesses).

## Category 6. Artificial Intelligence & IoT

### 38. Talk about artificial neural networks. Describe their structure. What tasks can ANN accomplish? What is the difference between classification and clustering? #ai #neuralnetworks
* **Краткий ответ**: ANN mimics the brain using input, hidden, and output layers to recognize, translate, or predict. Classification groups data into predefined classes; clustering finds undefined categories.
* **Пояснение**: An Artificial Neural Network (ANN) mimics the brain using input, hidden, and output layers connected by weights to handle tasks like recognition, translation, and prediction. Classification sorts data into predefined classes, whereas clustering groups it into undefined categories.

### 39. What is backpropagation? Why are neural networks compared to ‘black boxes’? #ai #neuralnetworks
* **Краткий ответ**: Backpropagation adjusts network weights backward using output error. Neural networks are "black boxes" because their internal decisions and weight dynamics are complex and hard to interpret.
* **Пояснение**: Backpropagation is a learning process that adjusts connection weights backward based on output errors, while neural networks are called "black boxes" because their internal decision-making remains hidden.

### 40. What is IoT? Where can the technology be implemented? What are the major problems in using the IoT technology? #iot
* **Краткий ответ**: A network of physical objects collecting/transferring data without human intervention. Major challenges include security risks (hacking) and managing the huge volume of devices and data.
* **Пояснение**: The Internet of Things (IoT) is a network of physical objects that can collect and transfer data without human interaction. The major problems are security risks (hackers stealing confidential data) and the difficulty of managing a massive number of devices and data.
